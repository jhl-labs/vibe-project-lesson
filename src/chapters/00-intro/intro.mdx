# Vibe Coding & Claude Code 교육

> AI Agent와 함께하는 현대적 소프트웨어 개발의 새로운 패러다임

## 이 교육에서 배우는 것

이 교육 자료는 **Vibe Coding**의 개념부터 **Claude Code**를 활용한 실전 개발 워크플로우까지 체계적으로 다룹니다.

<Callout type="info" title="교육 대상">
  AI 도구를 활용한 개발에 관심있는 모든 개발자를 대상으로 합니다.
  기본적인 프로그래밍 경험이 있으면 충분합니다.
</Callout>

## 교육 구성

| Part | 주제 | 내용 |
|------|------|------|
| **Part 1** | 기초 | 바이브 코딩, Claude Code, CLAUDE.md |
| **Part 2** | 프로젝트 템플릿 구조 | .agent 디렉토리, 컨텍스트, 아키텍처, 가이드라인 |
| **Part 3** | AI Agent 워크플로우 | 슬래시 커맨드, 프롬프트, 스킬, 서브에이전트 |
| **Part 4** | MCP와 통합 | Model Context Protocol, GitHub Actions |
| **Part 5** | 실전 예제 | TypeScript API, Python API |
| **Part 6** | 모범 사례 | 보안, 팀 협업, 효과 측정 |

## 학습 방법

1. **순서대로 학습**: Part 1부터 차례로 진행하세요
2. **실습 병행**: 코드 블록의 예제를 직접 따라해보세요
3. **퀴즈 풀기**: 각 챕터 끝의 퀴즈로 이해도를 점검하세요
4. **파일 뷰어 활용**: 실제 템플릿 파일 내용을 직접 확인하세요
