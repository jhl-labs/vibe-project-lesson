# Vibe Coding & Claude Code 교육

> AI Agent와 함께하는 현대적 소프트웨어 개발의 새로운 패러다임

## 이 교육에서 배우는 것

이 교육 자료는 **Vibe Coding**의 개념부터 **Claude Code**를 활용한 실전 개발 워크플로우까지 체계적으로 다룹니다.

<Callout type="info" title="교육 대상">
  AI 도구를 활용한 개발에 관심있는 모든 개발자를 대상으로 합니다.
  기본적인 프로그래밍 경험이 있으면 충분합니다.
</Callout>

## 교육 구성

| Part | 주제 | 내용 |
|------|------|------|
| **Part 1** | 기초 | 바이브 코딩, Claude Code, CLAUDE.md, 엔터프라이즈 보안, 프롬프트 엔지니어링 |
| **Part 2** | 프로젝트 템플릿 구조 | 프로젝트 지식 구조, 컨텍스트, 아키텍처, 가이드라인 |
| **Part 3** | AI Agent 워크플로우 | 슬래시 커맨드, 프롬프트, 스킬, 서브에이전트, Hooks |
| **Part 4** | MCP와 통합 | Model Context Protocol, GitHub Actions, MCP 서버 활용 |
| **Part 5** | 실전 예제 | TypeScript API, Python API, 프론트엔드, E2E 워크플로우 |
| **Part 6** | 모범 사례 | 보안, 팀 협업, 효과 측정 |

## 학습 방법

1. **순서대로 학습**: Part 1부터 차례로 진행하세요
2. **실습 병행**: 코드 블록의 예제를 직접 따라해보세요
3. **복습하기**: 각 챕터의 핵심 개념을 직접 실습해보세요
4. **파일 뷰어 활용**: 실제 템플릿 파일 내용을 직접 확인하세요

<Callout type="tip" title="실습 준비">
  교육을 최대한 활용하려면 다음을 준비하세요:
  - Claude Code 설치 (네이티브 설치 권장)
    - **macOS / Linux**: `curl -fsSL https://claude.ai/install.sh | bash`
    - **Windows**: `irm https://claude.ai/install.ps1 | iex`
  - `claude` 명령어 실행 후 인증 (Claude Pro/Max 구독 또는 API 키)
  - 이 프로젝트 템플릿 클론
</Callout>

<Callout type="warning" title="npm 설치는 더 이상 권장되지 않습니다">
  이전에 `npm i -g @anthropic-ai/claude-code`로 설치한 경우, `claude install` 명령어로 네이티브 설치로 마이그레이션하세요. 기존 설정 파일은 그대로 유지됩니다.
</Callout>
