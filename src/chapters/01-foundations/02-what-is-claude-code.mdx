# Claude Code란?

> Anthropic이 만든 터미널 기반 AI 코딩 에이전트

## 개요

**Claude Code**는 Anthropic이 개발한 **CLI(Command Line Interface) 기반의 AI 코딩 에이전트**입니다. 터미널에서 직접 실행하여 파일 시스템 접근, 코드 생성/수정, Git 작업, 테스트 실행 등 개발 워크플로우 전반을 지원합니다.

<Callout type="info" title="핵심 포인트">
  Claude Code는 단순한 코드 자동완성 도구가 아닙니다. 
  프로젝트의 전체 컨텍스트를 이해하고, 자율적으로 작업을 수행할 수 있는 **AI 에이전트**입니다.
</Callout>

## 핵심 기능

### 1. 파일 시스템 접근

Claude Code는 프로젝트 디렉토리 내의 파일을 직접 읽고 쓸 수 있습니다:

- 프로젝트 구조 파악
- 기존 코드 분석
- 새 파일 생성 및 기존 파일 수정
- 설정 파일 관리

### 2. CLAUDE.md 기반 프로젝트 컨텍스트

프로젝트 루트의 `CLAUDE.md` 파일을 자동으로 읽어 프로젝트 규칙과 컨벤션을 이해합니다.

### 3. MCP (Model Context Protocol)

외부 서버와의 연동을 통해 기능을 확장합니다:
- 파일 시스템, Git, GitHub API
- 데이터베이스 (PostgreSQL, SQLite)
- 브라우저 자동화 (Puppeteer)
- 외부 API 호출

### 4. Subagent (서브에이전트)

복잡한 작업을 여러 전문 에이전트에게 위임할 수 있습니다:
- 아키텍트 에이전트
- 보안 분석 에이전트
- 테스트 전략 에이전트
- 문서화 에이전트

### 5. Hooks & Plugins

특정 이벤트에 반응하는 자동화 스크립트를 설정할 수 있습니다.

### 6. 내장 슬래시 커맨드

| 커맨드 | 설명 |
|--------|------|
| `/init` | CLAUDE.md 자동 생성 |
| `/clear` | 대화 기록 초기화 |
| `/help` | 도움말 표시 |
| `/compact` | 컨텍스트 압축 |
| `/cost` | 토큰 사용량 확인 |
