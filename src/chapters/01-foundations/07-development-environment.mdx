# 개발 환경 설정

> Claude Code를 최적으로 활용하기 위한 IDE, 터미널, 설정 가이드

## IDE 통합

Claude Code는 터미널 CLI가 기본이지만, 주요 IDE와의 공식 통합을 제공합니다.

### VS Code 확장

Anthropic이 직접 개발하고 배포하는 공식 확장입니다.

<CodeBlock
  code={`# 설치 방법
# 1. VS Code에서 Cmd+Shift+X (Mac) / Ctrl+Shift+X (Windows/Linux)
# 2. "Claude Code" 검색 → Anthropic 퍼블리셔 확인 → 설치
# 3. Claude Code CLI가 사전 설치되어 있어야 합니다

# 또는 커맨드라인에서
code --install-extension anthropic.claude-code`}
  language="bash"
  filename="VS Code 확장 설치"
/>

**주요 기능:**
- 편집 전 변경 계획 검토
- 자동 편집 수락 (수동 확인 불필요)
- `@` 멘션으로 파일 참조
- 대화 히스토리 및 여러 탭/창 지원
- 실시간 파일 변경사항 확인
- 체크포인트 시스템 (Esc 두 번 또는 `/rewind`로 이전 상태 복원)

<Callout type="info" title="VS Code 확장 문서">
  상세한 사용법은 [Claude Code VS Code 문서](https://code.claude.com/docs/en/vs-code)를 참조하세요.
  확장은 [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code)에서 설치할 수 있습니다.
</Callout>

### JetBrains 플러그인

IntelliJ IDEA, PyCharm, WebStorm, PhpStorm, GoLand, Android Studio를 지원합니다.

<CodeBlock
  code={`# 설치 방법

# 2. 설치 후 IDE 재시작

/>

  상세 문서는 [Claude Code JetBrains 문서](https://code.claude.com/docs/en/jetbrains)를 참조하세요.
</Callout>

## 터미널 설정

### 키바인딩

Claude Code의 주요 키바인딩입니다.

| 키 | 기능 | 비고 |
|-----|------|------|
| **Enter** | 메시지 전송 | |
| **Shift+Enter** | 줄바꿈 (멀티라인) | `/terminal-setup` 실행 필요 |
| **Escape (2회)** | 이전 메시지로 되돌리기 (rewind) | |
| **Shift+Tab (2회)** | Plan Mode 전환 | 파일 수정 없이 탐색/계획만 |
| **Tab** | 파일/경로 자동완성 | |

<Callout type="warning" title="Shift+Enter 설정 필요">
  Shift+Enter 줄바꿈은 기본적으로 작동하지 않을 수 있습니다.
  `/terminal-setup` 명령을 실행하면 현재 터미널(VS Code, iTerm2, WezTerm 등)에 맞게 자동으로 설정됩니다.
  수동 대안으로 `\` + Enter를 사용할 수도 있습니다.
  키바인딩 커스터마이징은 [공식 문서](https://code.claude.com/docs/en/keybindings)를 참조하세요.
</Callout>

## 상태 줄 (Status Line)

터미널 하단에 현재 모델, 토큰 사용량, 비용 등을 실시간으로 표시할 수 있습니다.

<CodeBlock
  code={`# 상태 줄 설정 가이드 실행
/statusline

# 또는 .claude/settings.json에 직접 추가
# statusLine 객체로 커스텀 스크립트 지정 가능
# 스크립트는 JSON 입력을 받아 모델, 토큰, 비용, 워크스페이스 정보 표시`}
  language="text"
  filename="상태 줄 설정"
/>

<Callout type="tip" title="상태 줄 문서">
  상태 줄 설정에 대한 자세한 내용은 [Claude Code 상태 줄 문서](https://code.claude.com/docs/en/statusline)를 참조하세요.
</Callout>

## 알림 설정

Claude Code가 작업을 완료하고 입력을 대기할 때 알림을 받을 수 있습니다.

<CodeBlock
  code={`# 터미널 벨 알림 (전역 설정)
claude config set --global preferredNotifChannel terminal_bell

# Hooks로 플랫폼별 데스크톱 알림 설정
# macOS: osascript를 사용한 네이티브 알림
# Linux: notify-send
# Windows: PowerShell 알림`}
  language="bash"
  filename="알림 설정"
/>

<Callout type="info" title="알림 설정 문서">
  Hooks를 활용한 알림 설정은 [Claude Code Hooks 문서](https://code.claude.com/docs/en/hooks-guide)를 참조하세요.
</Callout>

## 멀티 계정/프로필 관리

여러 API 키나 설정을 전환해야 하는 경우:

<CodeBlock
  code={`# CLAUDE_CONFIG_DIR 환경 변수로 설정 디렉토리 분리
export CLAUDE_CONFIG_DIR=~/.claude-work
claude   # 회사 설정으로 실행

export CLAUDE_CONFIG_DIR=~/.claude-personal
claude   # 개인 설정으로 실행

# 셸 별칭으로 편리하게 사용
alias claude-work='CLAUDE_CONFIG_DIR=~/.claude-work claude'
alias claude-personal='CLAUDE_CONFIG_DIR=~/.claude-personal claude'`}
  language="bash"
  filename="멀티 프로필 관리"
/>

## 추천 초기 설정 순서

1. **Claude Code CLI 설치** (네이티브 설치 권장)
2. **IDE 확장 설치** (VS Code 또는 JetBrains)
3. **`/terminal-setup` 실행** (Shift+Enter 등 키바인딩 설정)
4. **프로젝트 루트에서 `/init` 실행** (CLAUDE.md 자동 생성)
5. **`/config`로 권한 및 기본 설정 조정**
6. **`/theme`로 테마 선택**
7. **필요 시 `/statusline`으로 상태 줄 설정**
