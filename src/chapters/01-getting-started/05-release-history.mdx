# Claude Code 릴리스 히스토리

> 12개월, 325+ 버전 — Research Preview에서 Agent Teams까지의 진화

## 개요

Claude Code는 2025년 2월 Research Preview로 시작하여, 불과 12개월 만에 **AI 코딩 에이전트의 표준**으로 자리잡았습니다. 주 평균 6회 릴리스라는 빠른 개발 속도를 유지하며, 터미널 기반 도구에서 멀티 에이전트 플랫폼으로 진화했습니다.

<Callout type="info" title="릴리스 단계 구분">
  Claude Code의 발전은 크게 4단계로 나뉩니다:
  **Phase 1** Research Preview (v0.2.x) → **Phase 2** GA (v1.0.x) → **Phase 3** Major Redesign (v2.0.x) → **Phase 4** Current (v2.1.x)
</Callout>

</Callout>
